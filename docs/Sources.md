#Starter Tutorials & Resources
- Official Docs: https://www.pygame.org/docs/
- YouTube: “Pygame platformer tutorial” by Clear Code or Tech With Tim
- Free assets: OpenGameArt.org → search “platformer tileset”

#Libraries:
- pillow for advanced image loading/manipulation
- numpy for math and physics logic if you extend your game later
- pygame
- random

#Things to learn or to consider
- How the background changes, this is what the camera sees
- Game loop, update and draw each fame (fps) 
- Groups: each element is an object, characters, adversaries, obstacles, prices, etc.
- Gravity and Jumping
- Tile map per level 
- Coins and prices 
- Collision detection
- Jumps and moving
